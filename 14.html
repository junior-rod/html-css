Fontes em CSS

Introdução

Para configurar a família tipográfica que será aplicada a um determinado texto, usamos a propriedade font-family das CSS. Se indicarmos mais de uma família na sequência, estamos indicando ao navegador que dê preferência para a primeira.

Prática

Escolher tipo de fonte

No seletor da tag que você deseja alterar, adicione: font-family: ; e escolha as fontes que você deseja. Exemplo: font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;

Tamanho de fontes

A W3C recomenda o uso do px ou em. A medida 'px' é referente a pixel e a medida 'em' é referente ao tamanho padrão de uma fonte.

16px=1em GERALMENTE.

Outros estilos

Existem outras formatações muito usadas em CSS, que são as propriedades font-style: ; para aplicar o itálico e font-weight: ; para aplicar o negrito.

Shorthand: font: ;

Shorthand é um "atalho" que existe em CSS para facilitar a configuração de determinadas coisas, por exemplo, é possível reduzir o font-family, font-size, font-style, font-weight em uma só propriedade:

font: ;

A ordem dos atributos de uma shorthand em CSS é MUITO importante. No caso da propriedade font, a ordem a ser seguida é:

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family

Exemplo:
font: italic bold 1em Arial, Helvetica, sans-serif;

Alinhamentos

Para mudar o alinhamento de um título, parágrafo ou qualquer outra coisa, é possível fazer isso em CSS.

Basta utilizar a propriedade text-align: ;

Como utilizar fonte do Google Fonts

Na página da fonte, no canto superior direito há 3 quadrados com um simbolo '+' (View selected families), clique nele e copie as linhas de código que estão entre a tag style para adicionar dentro da tag style (na primeira linha após a tag) da página ou dentro do arquivo .CSS. Após isso, volte no site do Google Fonts, e copie a linha de código que entrega o font-family:...; e copie dentro do seu style body, h1, h2, p...

Como utilizar fonte externa

Dentro das primeiras linhas da tag style, você vai digitar @font-face e escolher a opção que aparece a palavra 'Emmet'.

Em font-family: você dará o nome que você quiser dentro de duas aspas simples, ex: 'nome qualquer'

Já em url, você vai indicar onde sua(s) fonte(s) está(ão) no servidor ou seja, você pode adicionar uma virgula url() e adicionar outro formato da mesma fonte. Também, vai adicionar após cada url() o format() da fonte. Tipos de format:

  • opentype (.otf)
  • truetype (.ttf)
  • embedded-opentype
  • truetype-aat(Apple Advanced Typography)
  • svg

Feito os passos acima, basta quando for adicionar o atributo font-family em algum body, h1, h2... dentro da tag style, colocar o nome da fonte também dentro de aspas simples, por exemplo:
font-family: 'nomequalquer', Arial, Helvetica, sans-serif;
