Listas com HTML5

Listas Ordenadas

A HTML chama de ordered lists todas aquelas listas onde a ordem é algo muito importante.

Para criar uma lista ordenada, usa-se a tag <ol> para delimitar a lista e <li>(list item) para identificar cada item da lista.

Código: 

    <ol>
    <li>hehe</li>
    <li>haha</li>
    <li>hoho</li>
    </ol>

Resultado:

  1. hehe
  2. haha
  3. hoho

A tag ol pode possuir parâmetro type <ol type=""></ol>, onde é possível configurar o tipo de marcador da lista atual. Sendo o:

  • 1 - Padrão. Listas enumeradas (1,2,3...)
  • A - Listas alfabéticas maiúsculas (A,B,C...)
  • a - Listas alfabéticas minúsculas (a,b,c...)
  • I - Listas com algarismos romanos maiúsculos
  • i - Lista com algarismos romanos minúsculos.

Também é possível indicar o início da contagem usando o parâmetro start, exemplo: <ol type="I" start="5"></ol> vai gerar números romanos a partir de V.

  1. hehe
  2. haha
  3. hoho

Listas não Ordenadas

A HTML chama de unordered lists todas aquelas listas onde a ordem NÃO é algo muito importante.

Para criar uma unordered list, usa-se a tag <ul> para delimitar a lista e <li>(list item) para identificar cada item da lista.

O marcador padrão é a bolinha preenchida preta, mas existe a opção de configurar a propriedade type da tag <ul type=""> com os seguintes valores:

  • disc - Padrão. Bola preta toda pintada.
  • circle - Bola com bordas pretas sem preenchimento
  • square - Quadrado preto todo pintado

Lista de definição

É como se fosse um dicionário, temos os termos e as suas descrições.

Toda lista de definições está dentro de uma tag <dl></dl> (definition list). Cada termo é uma <dt> (definition term) e cada descrição é um <dd> (definition description). Entendeu? Pq eu não, mas segue o exemplo:

HTML
Linguagem de marcação utilizada para criar o conteúdo de sites.
CSS
Linguagem de marcação para a especificação de estilos em sites.
JavaScript
Linguagem de programação para criar interatividades em sites.

O que mais importa é o significado desse tipo de listagem, pois ele relaciona diretamente os pares Termo + Descrição e isso ajuda bastante com mecanismos de busca.
