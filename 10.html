Hyperlink

Para criar um hyperlink, você deve criar âncoras através da tag <a> </a>. Veja o exemplo:

<a href="https://www.youtube.com">Clique aqui para acessar o Youtube.</a>

Atributos de hyperlinks

O principal atributo dessa tag é o href=" ", que cria uma referência hipertexto e é nela que vai a Uniform Resource Locators(URL). Mas, como você já sabe, é possível adicionar mais de um atributo em uma tag, veja alguns exemplos bastante utilizados:

  • hreflang="en" indica o idioma do destino.
  • target="" controla onde o site destino irá abrir:
    • target="_self" abre o link na janela atual (padrão);
    • target="_blank" abre o link em uma nova guia/aba;
  • rel="" indica qual é a natureza do destino:
    • rel="next" indica que o link é para a próxima parte do documento atual;
    • rel="prev" indica que o link é para a parte anterior do documento atual;
    • rel="author" indica que é um link para o site do autor do artigo atual;
    • rel="external" indica que é um link para outro site que não faz parte do site atual;
    • rel="nofollow" indica que é um link para um site não endossado, como um link pago.

Downloads

Para fazer download, utiliza-se a tag <a> </a>, porém com algumas alterações.

<a href="arquivos/livros/meulivro.pdf" download="meulivro.pdf" type="application/pdf">Clique aqui para fazer o download do meu livro.</a>
