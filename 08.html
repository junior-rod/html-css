Agora a coisa muda

Negrito

A partir de agora, quando você quiser algo em negrito, utilize a tag <strong> </strong> porque ela significa que o termo possui força dentro da frase.

Itálico

Para deixar alguma parte do texto em itálico, utilize a tag <em> </em>, que significa emphasis (ênfase), já que a ideia do itálico é dar ênfase a algo.

Marca texto

Para fazer marcas de marca texto em HTML5, basta utilizar a tag <mark> </mark>. Essa tag serve para trechos que você deseja dar uma ênfase descomunal.

Big e small

A tag <big> </big> que servia para deixar o texto maior foi de base, mas a tag <small> </small> que serve para deixar o texto pequeno, principalmente em rodapé e legendas de fotos, ainda está viva.

Texto taxado

Para taxar o texto use a tag <del> </del>. Ela significa que o texto está ali, pode ser lido, mas deve ser desconsiderado pelo leitor. S I G N I F I C A D O!

Texto inserido (sublinhado)

O texto inserido significa que o trecho deve ser lido e você DEVE prestar atenção nele. Para utilizar o texto inserido, basta usar a tag <ins> </ins>

Sobrescrito e Subscrito

Para textos sobrescritos (ex: H2O) e subscritos (ex: m2), basta utilizar a tag <sub> </sub> para textos sobrescritos e <sup> </sup> para subscritos.

Citações

Para criar uma citação em HTML, a primeira forma é utilizar a tag <q> </q> (quote). O texto que estiver entre essa tag vai receber automaticamente as aspas, mas não terá deslocamento, ideal para citações no meio do parágrafo.

Para citações que tenham um parágrafo só pra si, usa-se a tag <blockquote> </blockquote>. O texto ganha um recuo automático e pode usar o parâmetro cite dentro da tag para adicionar o link do local de onde foi retirado.
