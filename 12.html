Trabalhando com estilos (CSS)

CSS inline style

Em ordem de prioridade que o código fará, esse método sera o prioritário, ou seja, ele sempre levará esse em consideração. Para utilizar CSS inline basta adicionar o parâmetro style dentro da tag, por exemplo:

<body style="font-family: Arial, Helvetica, sans-serif; color: blue;">

Isso fará com que todo o body fique com uma fonte sem serifa e com as letras azuis.

CSS internal style

Para aplicar estilos de forma mais dinâmica e prática, adicione uma tag <style> dentro da área <head>

Seu código deve ficar mais ou menos assim: 

                <html>
                <head>
                ...
                <style>
                    

                        body{
                            font-family: Arial, Helvetica, sans-serif;
                            background-color: gray;
                            color: white;
                        }
                        h1{
                            color: green;
                        }
                    

                </style>
                </head>
                <body>
                </body>
                </html>

CSS external style

utilizamos a tag <link> especialmente configurada para trabalhar com arquivos externos de estilo. Essa tag deve ser colocada dentro da área <head>. Basta digitar 'link' no head e escolher a opção link:css.

Após isso, segure a tecla ctrl e clique no nome do documento, dessa forma o VS Code vai criar o arquivo .css para você.

ATENÇÃO! Ao acessar o arquivo .css a primeira coisa que deve ser feita é adicionar @charset "UTF-8"; para evitar problemas com acentuação.

Para utilizar essa mesma configuração montada dentro do arquivo .css em outras páginas, basta adicionar o link no head e escolher o mesmo nome de arquivo que você escolheu antes (É OBVIO QUE O ARQUIVO .CSS PRECISA TÁ DENTRO DA MESMA PASTA DA PÁGINA).
